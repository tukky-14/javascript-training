### 定数・変数

```javascript
const test = ''; // 定数（厳密には定数ではないが、定数のようなもの）
let test1 = ''; // 変数
var test2 = ''; // 使用したらあかん！

// test = '1' // × 再代入不可
test1 = '1'; // ● 再代入可能
```

<br/>

### データ型

```javascript
const string = 'test'; // 文字列
const number = 123; // 数値
const boolean = true; // 真偽値（if 文とかに使う）
```
