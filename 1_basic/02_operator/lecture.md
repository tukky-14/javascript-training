### 条件式と演算子

```javascript
const num = 1;
if (num === 1) {
    console.log('numは1です');
}

const num2 = 2;
if (num2 % 2 === 1) {
    console.log('num2は奇数です');
} else {
    console.log('num2は偶数です');
}
```

<br/>

### 三項演算子

```javascript
num2 % 2 === 1 ? console.log('num2 は奇数です') : console.log('num2 は偶数です');
```
